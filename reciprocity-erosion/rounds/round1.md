# Source: Tidewatch Project Overview (internal memo, 2026)

The Tidewatch project deploys a network of autonomous buoy sensors along the
Coral Shelf to monitor coastal salinity in real time. It is operated by the
Marren Lab. All sensor readings are transmitted using the DriftNet protocol.
The project's headline result to date is the Salinity-Drift Finding: a strong
correlation between tidal drift velocity and near-surface salinity. Tidewatch
buoys are calibrated in the field using the Kelp Calibration method before each
deployment season.
